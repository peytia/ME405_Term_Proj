def motor_run.move_pitch (   motor,
  degrees 
)

Moves the pitch motor by a specified number of degrees.

Updates and reads the pitch motor encoder value, sets the controller setpoint to the desired number of degrees, runs the controller to calculate and set the duty cycle. The input and output is converted between gun rotation degrees and motor encoder tics. This method is meant to be run in a loop, continuously updating the pitch desired position

Parameters
motorThe motor object that is to be controlled
degreesThe desired end pitch position of the gun
Returns
The current motor pitch position, in gun degrees
